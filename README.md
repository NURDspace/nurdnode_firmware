# Switches
Switch 10 switches between Autonomic mode and DMX mode
* On Autonomic
* Off DMX

The + and - Positions of the dip switches both enable the dipswitch
* - == 1
* + == 1
* center == 0

## Autonomic mode
Switch        | Function
------------- | -------------
1             | Bezerk without led
2             | Random color
3             | Fade on/off
4             | Strobe mode
